# 天使与恶魔

一个可本地运行的网页小游戏，实现了适合朋友对局的“短局版”天使与恶魔。

## 本地运行

直接用浏览器打开 `index.html` 即可。

推荐用本地服务器预览：

```bash
npm start
```

然后访问 `http://127.0.0.1:8000`。

## 本地联机

1. 房主运行服务器：

```bash
node server.mjs
```

2. 房主打开 `http://127.0.0.1:8000`，点击“创建房间”。房主会成为恶魔。

3. 把页面显示的 5 位房间码发给朋友。朋友打开同一个服务器地址，输入房间码后点击“加入”。朋友会成为天使。

4. 恶魔每回合连续封锁 2 格；天使回合移动 1 格。服务器会校验身份和规则。

如果朋友和你在同一个局域网，需要让服务器监听局域网地址：

```powershell
$env:HOST="0.0.0.0"; node server.mjs
```

然后把你的电脑局域网 IP 发给朋友，例如 `http://192.168.1.23:8000`。

也可以发带房间码的链接：

```text
http://你的服务器地址:8000/?room=房间码
```

## 玩法

- 设置天使力量 `K`、逃脱距离、最大天使回合和棋盘半径后开始游戏。
- 恶魔先手，点击空格放置一个路障。
- 天使回合点击高亮范围内的空格移动，八方向相邻均算一步。
- 天使移动途中可以穿过路障，但落点不能是路障。
- 天使到达绿色逃脱外圈，或撑过最大天使回合，天使获胜。
- 天使没有任何合法落点时，恶魔获胜。

当前默认参数：

- 棋盘半径 `20`，也就是 `41x41`
- `K = 1`
- 逃脱距离 `22`
- 恶魔每回合封锁 `2` 格
- 最大天使回合 `32`
- 天使禁止回到上一格

说明：`41x41` 棋盘从中心到边界是 20 格，逃脱距离 `22` 表示天使需要突破边界外的绿色逃脱线才算获胜。
